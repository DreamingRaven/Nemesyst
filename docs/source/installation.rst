Installation
============

Automated
*********

Generic
+++++++

In future you will be able to:

.. code-block:: bash

  pip install nemesyst

Archlinux
+++++++++

Install `nemesyst-git <https://aur.archlinux.org/packages/nemesyst-git/>`_:sup:`AUR`.

Manual
******

.. code-block:: bash

  git clone https://github.com/DreamingRaven/nemesyst
  cd nemesyst
  python setup.py install
